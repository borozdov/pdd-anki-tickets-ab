## Вопрос 1
**С какой максимальной скоростью можно продолжить движение за знаком?**

![img](https://s.drom.ru/i24227/pdd/tickets/2016/1542608233.jpg)

1. 60 км/ч
2. 50 км/ч
3. 30 км/ч
4. 20 км/ч

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. 20 км/ч


<b>Правильный ответ: 4</b>
<br/>
            В жилой зоне, согласно знаку, в которую Вы въезжаете, разрешается движение со скоростью не более 20 км/ч.
(«Дорожные знаки» <a href="https://www.drom.ru/pdd/pdd/sign_5_21/" target="_blank">5.21</a>, пункт <a href="https://www.drom.ru/pdd/pdd/punkt_10_2/#p10_2" target="_blank">10.2</a> ПДД).

## Вопрос 2
**Какие из перечисленных действий запрещены водителям механических транспортных средств в жилой зоне?**

1. Сквозное движение
2. Учебная езда
3. Стоянка с работающим двигателем
4. Все перечисленные действия

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. Все перечисленные действия


<b>Правильный ответ: 4</b>
<br/>
            Сквозное движение, учебная езда, стоянка с работающим двигателем в жилой зоне и на дворовых территориях запрещены.
(Пункты <a href="https://www.drom.ru/pdd/pdd/punkt_17_2/#p17_2" target="_blank">17.2</a>, <a href="https://www.drom.ru/pdd/pdd/punkt_17_4/#p17_4" target="_blank">17.4</a> ПДД)        
</details>

## Вопрос 3
**Где могут двигаться пешеходы в жилой зоне?**

1. Только по тротуарам
2. По тротуарам и в один ряд по краю проезжей части
3. По тротуарам и по всей ширине проезжей части

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 3. По тротуарам и по всей ширине проезжей части


<b>Правильный ответ: 3</b>
<br/>
            В жилой зоне, на дворовых территориях пешеходы имеют преимущество перед механическими транспортными средствами. Их движение разрешается как по тротуарам, так и по всей ширине проезжей части.
(Пункты <a href="https://www.drom.ru/pdd/pdd/punkt_17_1/#p17_1" target="_blank">17.1</a>, <a href="https://www.drom.ru/pdd/pdd/punkt_17_4/#p17_4" target="_blank">17.4</a> ПДД)        
</details>

## Вопрос 4
**С какой максимальной скоростью разрешается движение транспортных средств в жилых зонах, велосипедных зонах и на дворовых территориях?**

1. 10 км/ч
2. 20 км/ч
3. 40 км/ч

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 2. 20 км/ч


<b>Правильный ответ: 2</b>
<br/>
            Движение в жилой зоне, на дворовых территориях имеет ряд ограничений. К таким ограничениям относится и скорость, которая не должна превышать 20 км/ч.
(Пункт <a href="https://www.drom.ru/pdd/pdd/punkt_10_2/#p10_2" target="_blank">10.2</a> ПДД)        
</details>

## Вопрос 5
**При выезде из жилой зоны необходимо уступить дорогу:**

1. Только транспортным средствам с включенным проблесковым маячком
2. Только транспортным средствам, приближающимся слева
3. Только транспортным средствам, приближающимся справа
4. Всем транспортным средствам

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 4. Всем транспортным средствам


<b>Правильный ответ: 4</b>
<br/>
            При выезде из жилой зоны водители должны уступить дорогу другим участникам движения, в том числе всем транспортным средствам.
(Пункт <a href="https://www.drom.ru/pdd/pdd/punkt_17_3/#p17_3" target="_blank">17.3</a> ПДД)        
</details>

## Вопрос 6
**Водители должны уступать дорогу другим участникам движения:**

1. При выезде из жилой зоны
2. При выезде с дворовой территории
3. В обоих перечисленных случаях

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 3. В обоих перечисленных случаях


<b>Правильный ответ: 3</b>
<br/>
            При выезде из жилой зоны, дворовой территории водители должны уступать дорогу другим участникам движения. Правильный ответ - в обоих перечисленных случаях.
(Пункты <a href="https://www.drom.ru/pdd/pdd/punkt_17_3/#p17_3" target="_blank">17.3</a>, <a href="https://www.drom.ru/pdd/pdd/punkt_17_4/#p17_4" target="_blank">17.4</a> ПДД)        
</details>

## Вопрос 7
**Вы обязаны уступить дорогу грузовому автомобилю:**

![img](https://s.drom.ru/i24227/pdd/tickets/2016/1542609084.jpg)

1. Только при повороте направо
2. Только при повороте налево
3. В обоих перечисленных случаях

<details>
<summary>Показать ответ</summary>

**Правильный ответ:** 3. В обоих перечисленных случаях


<b>Правильный ответ: 3</b>
<br/>
            Вы выезжаете из жилой зоны, о чём информирует знак <a href="https://www.drom.ru/pdd/pdd/sign_5_22/" target="_blank">5.22</a> «Конец жилой зоны». Уступаете дорогу другим участникам дорожного движения, в данной ситуации - грузовому автомобилю в обоих перечисленных случаях.
(«Дорожные знаки», пункт <a href="https://www.drom.ru/pdd/pdd/punkt_17_3/#p17_3" target="_blank">17.3</a> ПДД)        
</details>
